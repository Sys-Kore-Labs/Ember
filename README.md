# Ember

Ember is a burnout and apathy tracking tool designed to help users recognize warning signs before reaching a breaking point.

## Why I built this

I experienced burnout myself and realized how difficult it can be to notice the warning signs early enough.

Ember is my attempt to create a simple tool that helps people track their sleep, energy, stress, motivation, and recovery over time.
